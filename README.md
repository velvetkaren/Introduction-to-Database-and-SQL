# Introduction-to-Database-and-SQL
Introduction to the basics of how SQL solutions can be formulated  
Learned the steps for MySQL installation on MacOS and Windows  
Understand how to SELECT and manipulate specific data from tables using keywords like SELECT, LIMIT, DISTINCT, INSERT, UPDATE, and DELETE  
Understand filtering of data can be carried out using keywords like WHERE/HAVING, LIKE and its types, and between  
Understand aggregation of data using ORDER BY, GROUP BY case when and various aggregate functions like SUM, COUNT, AVG, MIN, and MAX  
Understand the concept of joins, inner joins, LEFT, RIGHT, FULL, SELF, UNION and UNION ALL
Lastly, looked into the order of execution of SQL commands which is FROM, WHERE, GROUP BY, HAVING, SELECT, ORDER BY, and LIMIT
